ZENDS AI Customer Support Copilot is a Retrieval-Augmented Generation (RAG) based intelligent assistant built for a virtual telecom company.

The system analyzes customer queries, classifies their intent and sentiment, and retrieves accurate answers from official company documents using semantic similarity search.

This project demonstrates:

Document-grounded AI responses

Embedding-based retrieval using FAISS

Intent and sentiment classification

Interactive Streamlit dashboard
